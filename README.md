# Daikin AC Controller – Android App
**For Redmi Note 8 | ARC484B32 Remote | No Ads | IR Blaster**

---
## How to build and get the APK (step-by-step for beginners)

### Step 1 – Create a free GitHub account
1. Open https://github.com in your phone browser or PC
2. Click **Sign up** → enter email, password, username → Verify → Done

### Step 2 – Create a new repository
1. Click the big **+** button (top-right) → **New repository**
2. Repository name: `daikin-ac`
3. Leave everything else default → Click **Create repository** (green button)

### Step 3 – Upload the files
1. On the empty repo page, click **uploading an existing file**
2. Drag and drop ALL files/folders from the zip I gave you
3. Click **Commit changes** (green button)

### Step 4 – Wait for the build (~5 minutes)
1. Click the **Actions** tab at the top
2. You'll see "Build Daikin AC APK" running (yellow circle = building)
3. Wait for it to turn GREEN ✅

### Step 5 – Download your APK
1. Click on the green build → scroll down to **Artifacts**
2. Click **Daikin-AC-Controller-APK** to download the zip
3. Extract the zip → you get `app-release.apk`

### Step 6 – Install on Redmi Note 8
1. Transfer `app-release.apk` to your phone via USB or WhatsApp
2. On your phone: Settings → Additional Settings → Privacy → Install unknown apps → Allow
3. Open the APK file → Install → Open
4. Point your phone's **top edge** at the AC (IR blaster is there on Redmi Note 8)
5. Press **TURN ON** – you should hear the AC beep!
